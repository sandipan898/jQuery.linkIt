# LinkIt jQuery Plugin
**LinkIt** is a simple jQuery plugin to attach links to elements

## Version
0.3.0

## Usage 
    $(document).ready(function() {
            $('span').linkIt({
                href       :'http://test.com',
                text       :'Click me',
                target     :'_blank',
            });
        });

## Vendors
